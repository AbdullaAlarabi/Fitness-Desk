Fitness Desk — Approved Exercise Demo Media (24 items)

This package contains the final approved source exercise demo posters for Fitness Desk.

Copy this folder into the app:
public/media/training/exercises/demo/

The files are named with the final Codex-ready filenames. Most source posters are 1024 × 1536 or very close 2:3 vertical. Use object-fit: contain in the demo modal.

Thumbnails are not included. Generate 512 × 512 thumbnails from these approved posters after implementation.

Important locked corrections:
- Lying Leg Curl Machine replaces the earlier seated leg curl direction.
- Do not use grid/collage images.
- Do not map Cable Crunch to Pallof Press.
- Do not map Reverse Pec Deck to Pec Deck / Chest Fly.
- Do not map Face Pull to Reverse Pec Deck.
- Do not map Straight-Arm Cable Pulldown to Rope Triceps Pushdown.

See manifest/approved_exercise_media_manifest.json for the exact mapping.
